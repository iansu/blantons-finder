# Blantons Finder

[![Build status](https://github.com/iansu/blantons-finder/workflows/CI/badge.svg)](https://github.com/iansu/blantons-finder/actions)

Find all the Blantons in BC

## Usage

### `yarn start`

## Building

### `yarn build`

To clean the build directory run `yarn clean`

## Deploying

### `yarn deploy`
